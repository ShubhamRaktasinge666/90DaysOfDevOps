<h1>Process Management Commands</h1>
<p><b>ps</b>: Displays a snapshot of current active processes.</p>
<p><b>top</b>: Show all top process which are running and how much consume resources.</p>
<p><b>kill</b>: It is a command which kills the process according to it's process id.</p>
<p><b>jobs</b>: Lists all active background and suspended jobs in the current shell session.</p>
<p><b>nohup</b>: Runs a command that will continue even after the user logs out or the terminal is closed.</p>
<p><b>free</b>: : Displays the amount of free and used memory in the system</p>

<h1>File System Commands</h1>
<p><b>ls</b>: Displays / List the all files and directory in the current present working directory.</p>
<p><b>who</b>: Displays the current login information of a user.</p>
<p><b>whoami</b>: Displays a current login user.</p>
<p><b>cd</b>: It is used to change the directory or to go from one directory to another.</p>
<p><b>uname</b>: Displays the current kernel / platform version.</p>
<p><b>lscpu</b>: Displays the cpu utilization.</p>
<p><b>man</b>: Displays the manual of the command.</p>
<p><b>history</b>: Displays all the command which are previously used.</p>
<p><b>touch</b>: create an empty file.</p>
<p><b>cat</b>: Displays a content in a file.</p>
<p><b>echo</b>: Displays as it is content which are writeen in side quotes( "" ).</p>

<h1>Networking Commands</h1>
<p><b>ping</b>: Test the connectivity between devices / receiving the packets which shows that the connectivity is fine.</p>
