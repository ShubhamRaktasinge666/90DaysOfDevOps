# <h1>Read and Write a files in linux</h1>
* `touch notes.txt ` <br>
  Creating a empty text file called  notes.txt<br>
* `echo "I am shubham Raktasinge" > notes.txt ` <br>
  Write a content in file<br>
* `echo "Cloud and Devops enthusiast" >> notes.txt ` <br>
* `echo "This is day 6 task" >> notes.txt ` <br>
  Append the content into the notes.txt file<br>

* `cat notes.txt ` <br>
  Read the file notes.txt<br>

* `cat notes.txt | head -n 1` <br>
Read the first line of notes.txt<br>
* `cat notes.txt | tail -n 2` <br>
Read the last two line of notes.txt<br>

* ` echo "Learning the linux commands" | tee -a notes.txt ` <br>
Write using tee command that also prints the output to terminal -a appends
